# Media & Entertainment News Automation

Automated daily scraper that extracts entertainment news & TV updates 
from 3 sources, prevents duplicates, and sends email reports.

## Features
✅ 3 daily extractions (11 AM, 3 PM, 6 PM)
✅ Smart duplicate detection
✅ Email notifications
✅ Excel exports
