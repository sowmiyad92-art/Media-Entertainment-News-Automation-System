# Setup Instructions - Step by Step

## Prerequisites
- Windows 10/11
- Python 3.10+
- Gmail account

## Installation

### Step 1: Clone Repository
```bash
git clone https://github.com/yourusername/media-automation.git
cd media-automation
```

### Step 2: Create Virtual Environment
```bash
python -m venv venv
venv\\Scripts\\activate
```

### Step 3: Install Dependencies
```bash
pip install -r requirements.txt
```

git add .
